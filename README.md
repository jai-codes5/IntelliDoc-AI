# 🚀 IntelliDoc-AI

> AI-powered Document Intelligence System built using Flask and Google Gemini API for intelligent document analysis and summarization.

---

## 📖 Project Overview

IntelliDoc-AI is an AI-powered document analysis application that allows users to upload PDF or TXT documents and receive intelligent summaries using Google's Gemini AI.

The application provides a clean dashboard where users can upload documents, process them using AI, and quickly understand the important information without reading the entire document manually.

---

# ✨ Features

- 📄 Upload PDF and TXT documents
- 🤖 AI-powered document analysis using Google Gemini
- 📝 Automatic document summarization
- ⚡ Fast document processing
- 🎨 Modern responsive dashboard
- 🔒 Secure file upload validation
- 📊 Interactive user interface
- ❌ Error handling and validation

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/jai-codes5/IntelliDoc-ai.git
```

```bash
cd IntelliDoc-ai
```

---

## Create Virtual Environment

```bash
python -m venv venv
```

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 🔑 Environment Variables

Create a `.env` file.

```env
GEMINI_API_KEY=YOUR_API_KEY
SECRET_KEY=YOUR_SECRET_KEY
```

---

# ▶️ Run Application

```bash
python app.py
```

Open browser
